# Neel Maheshwari Mechanical Engineering Portfolio

Professional portfolio site built with Next.js, React, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Keep the default Next.js settings.
4. Deploy.

## Project photos

Replace `public/images/engineering-hero.png` or add new images under `public/images/`, then update the `projects` array in `app/page.tsx`.
